#  KDoc-to-TS Gradle Plugin

This plugin automatically embeds your KDoc comments into the TypeScript definitions (`.d.mts` or `.d.ts`) generated by the Kotlin/JS compiler.

Because Kotlin/JS can't.

## Usage
This plugin requires KSP.

In all modules targeting Kotlin/JS, add the KSP symbol processor to the `kspJs` configuration
```kotlin
dependencies {
    add("kspJs", "io.github.justincodinguk.safeexports:kdoc-export-ts-processor:1.0.0")
}
```


Enable the `embedKdoc` flag in module to be exported as the final library

```kotlin
kDocExport {
    embedKdoc = true
}
```

Run the following task to embed your KDocs inside the generated TypeScript definitions.
```bash
./gradlew :your-module-name:embedTypeScriptDocs
```
